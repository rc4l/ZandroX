SOURCE CODE
-----------

was built from and the build scripts, is published at:

its version. This satisfies GPL-3.0 section 6(d); if that location ever becomes

COMPONENT LICENCES
------------------
The original Doom source was released by id Software under the GPL v2. ZandroX
distributes it under the GPL v3, the same position the GZDoom project takes; see
src/zandronum/docs/licenses/README.TXT for the detail and the caveat.

This tree descends from Zandronum, which forked ZDoom at 2.8pre / GZDoom 1.8.6 —
before GZDoom's own 2017 relicensing — so the relicensed upstream files were not
inherited and the headers here were corrected directly.

Zandronum's and Skulltag's own additions are under BSD-3-clause-style licences
with an added source-availability clause, reproduced at src/zandronum/LICENSE.txt
and src/zandronum/docs/Skulltag-license.txt.

Note on OpenSSL: version 3.x is Apache-2.0, which is compatible with GPL-3.0.
The older OpenSSL 1.x licence, with its advertising clause, was not — so this
distribution must stay on OpenSSL 3.x or later.

FMOD, which upstream Zandronum uses for audio, is proprietary and is NOT part of
this distribution; it has been replaced by the OpenAL stack listed above.
